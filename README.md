# Wedding Photos Link

This repo is set up so one GitHub Pages link opens the full wedding PDF as a clean, scrollable page on phone/desktop.

## Publish

1. Create a GitHub repo.
2. Upload `index.html`, `weddingpix_parts.json`, and all `weddingpix_part*.pdf` files.
3. In GitHub repo settings: `Pages -> Deploy from a branch -> main / (root)`.
4. Share the Pages URL.
